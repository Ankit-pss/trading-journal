# Trading Journal

A full-stack web application designed for traders to log, track, and analyze their trading performance.

## Project Structure

- `client/`: Contains the frontend user interface (HTML, JS).
- `server/`: Contains the Python backend application.

## Prerequisites

- Python 3.x

## Setup & Running

1. **Backend Setup:**
   Navigate to the project root and activate the virtual environment:
   ```bash
   source venv/bin/activate
   ```
   *(If dependencies are not installed, install them using `pip install -r requirements.txt` if available)*

2. **Run the Server:**
   ```bash
   cd server
   python app.py
   ```

3. **Frontend:**
   Simply open `client/index.html` in your web browser, or use a live server extension.
